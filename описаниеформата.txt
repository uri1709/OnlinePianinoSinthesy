Вот обновленное, максимально точное описание формата JSON для сервиса Piano Studio Pro с учетом жесткого правила наименования нот.
------------------------------
## Полная спецификация формата JSON для Piano Studio Pro
Формат данных представляет собой плоский массив JSON-объектов. Каждый объект описывает одну ноту, её положение на временной сетке, длительность и привязанный инструмент.
## 1. Правило наименования нот (note)
Название ноты — это строго определенная текстовая строка, которая состоит из двух частей: названия звука и номера октавы (без пробелов, дефисов или нижних подчеркиваний).
Для текстовой части разрешено использовать только следующие 12 базовых названий в верхнем регистре (капсом):

* ДО
* РЕБИМОЛЬ
* РЕ
* МИБИМОЛЬ
* МИ
* ФА
* СОЛЬБИМОЛЬ
* СОЛЬ
* ЛЯБИМОЛЬ
* ЛЯ
* СИБИМОЛЬ
* СИ

Пример записи: Нота Ля в третьей октаве записывается строго как "ЛЯ3", а Ми-бемоль во второй октаве — строго как "МИБИМОЛЬ2".

------------------------------
## 2. Временная сетка и позиционирование (numTact, pos)

* numTact (Целое число): Порядковый номер музыкального такта (начиная с 1). Задает глобальный временной отрезок.
* pos (Число от 0 до 3.99): Стартовая позиция ноты внутри указанного такта. Вся сетка построена на размере 4/4, где 1.0 равен одной четвертной доле:
* 0 — самое начало такта (первая доля).
   * 1.0 — вторая доля.
   * 2.0 — ровно середина такта (третья доля).
   * 3.5 — последняя восьмая доля такта.

------------------------------
## 3. Длительность звука (length)

* length (Число с плавающей точкой): Длина нотной полосы, измеряемая в четвертных долях:
* 0.5 — восьмая нота.
   * 1.0 — четвертная нота.
   * 2.0 — половинная нота.
   * 4.0 — целая нота (звучит весь такт).

------------------------------
## 4. Настройки синтезатора (idInstrument, nameInstrument)

* idInstrument (Целое число): MIDI-индекс инструмента (для стандартного пианино используется 0).
* nameInstrument (Строка): Название звукового пресета (например, "Acoustic Grand Piano").

------------------------------
## Пример готового объекта ноты:

{
  "numTact": 1,
  "pos": 1.5,
  "note": "СИБИМОЛЬ3",
  "length": 0.5,
  "idInstrument": 0,
  "nameInstrument": "Acoustic Grand Piano"
}

Если этот вариант описания полностью соответствует логике вашего движка, выберите следующее действие:

* Сгенерировать шаблон пустого такта для удобного ручного заполнения?
* Перейти к разбору новых музыкальных фрагментов по этой схеме?
